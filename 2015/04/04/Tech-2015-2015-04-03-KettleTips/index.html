Kettel 系统配置文件

在第一次运行kettle之后，kettle会在%HOME_USER_FOLDER%菜单里面创建一个 .kettle文件夹，

windows C:\Documents and Settings\${your user name}.kettle\kettle.properties
linux /home/${your user name }/.kettle/kettle.properties
